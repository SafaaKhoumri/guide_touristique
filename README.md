## 🛠️ Technologies
- React + Vite
- Netlify (déploiement)

## ✨ Fonctionnalités
- Page d'accueil avec hero et tours populaires
- Liste des circuits touristiques (Bruxelles)
- Page About du guide
- Page Reviews clients
- Page Contact avec carte Google Maps
- Bouton WhatsApp intégré
- Design responsive

## 🚀 Lancer le projet
```bash
npm install
npm run dev
```

## 📦 Build
```bash
npm run build
```
